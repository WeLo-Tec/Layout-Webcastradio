Webcastradio3 ist ein Standard 3 Spalten Layout inkl. Slider.

Besonderheiten:
Auf der Startseite wird direkt unter dem Slider der Radiostatus angezeigt
Der Radiostatus ist optimiert für einen laut.fm und einen SHOUTcast 2 Stream.
Im Footer ist auch noch eine navbar.

Der Radiostatus kann über Colorpicker im Admin farblich verändert werden.
Über den Admin kann man Moderatorenbilder uploaden, die im Radiostatus angezeigt werden.
Es werden auch die Coverbilder vom Titel angezeigt.
Es werden die Hörerzahlen und bei einem SHOUTcast auch noch die Bitrate angezeigt.
Zusätzlich ist eine Gruß- und Wunschbox integriert und auch über das Admin konfigurierbar.
